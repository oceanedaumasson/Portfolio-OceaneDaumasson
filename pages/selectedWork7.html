CALM VS CHAOS

GITHUB

Max/MSP

A two-part generative audio-visual project built in Max/MSP, exploring emotional contrast through real-time systems. The project investigates how sound, motion, and form can shape mood and perception across two contrasting environments.

Key Features

  • Real-time generative visual system
  • Particle dynamics and chaotic attractor
  • Audio-reactive parameter mapping
  • Timed randomization
